Modular Hardware Packages

Each subsystem in our training kit can be ordered as standalone modules or combined in one multi-functional bench system.

Module A

PLC Controller Panel

Teaches logic structures, binary gates, timer configurations, analog signal routing, and safety limits. Equipped with standard logic indicators and wiring ports.

  • Logic Processor: Siemens S7-1200 CPU 1212C DCDCDC
  • Digital I/O Ports: 8 Inputs (24V), 6 Relay Outputs
  • Analog Channels: 2 Inputs (0-10V potentiometer), 1 Output
  • Logic Software: Siemens TIA Portal V18 / STEP 7 Basic
Module B

HMI Control Interface

Teaches graphic telemetry design, system toggle programming, alarms triggers, and network status panel bindings. Fits directly onto standard DIN mounts.

  • Touch Display: 7.0" Color TFT LCD, 65k Colors
  • Interface Link: PROFINET Industrial Ethernet, RS-485
  • Logic Software: WinCC Basic (TIA Portal) / EasyBuilder Pro
  • Features: Active historical trend graphs, vector symbols
Module C

Sensor Feedback Array

Trainees learn about circuit logic switches and calibrate proximity thresholds using multiple industry-standard digital and analog sensor models.

  • Sensor Selection: Inductive, Capacitive, Photoelectric, Ultrasonic
  • Feedback Encoder: Incremental Rotary Encoder (2-ch, 100 PPR)
  • Limit System: 4x Roller micro limit switches
  • Network Link: IO-Link Master V1.1 integration hub
Module D

Electro-Pneumatic Actuators

Teaches motion sequence logic, pressure limit calibrations, pneumatic wiring grids, and double-acting cylinder driving.

  • Actuators: 2x Double-acting cylinders, magnetic reed limit switches
  • Flow Valves: 2x 5/2-Way Double Solenoid Valves (24V DC control)
  • Regulators: Filter regulator with analog bar dial gauge
  • Tube Fittings: 6mm instant push-fit plastic hose connectors
